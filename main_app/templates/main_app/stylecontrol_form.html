Style Help

Defaults

Background color: #19181A
Font color: #FAB139
Font opacity (%): 70
Font size: 18px

The default font-family is Source Sans Pro; the default font-size is 18px. In the font family field below, you can choose a new font from the drop down list. You can also enter a number in the font size field to make the fonts smaller or larger.

Choosing a new color is more complex. For example, if you wanted to make your website's text purple, you could enter into the font color field any of the following:

  • purple
  • #800080
  • rgba(128,0,128)

Many colors are available by name, as in the first option. The second option ("hexadecimal" notation) and the third option (called either "rgb()" or "rgba()" notation) allow for thousands of choices. This website is a great and simple tool for finding the specific shade you might want; coolors.co is even more powerful, but a little harder to use. Both helpfully generate hex and rgba codes for you to use.

If you want to revert to the default font color, you can copy and paste rgba(250,177,98,.7) into the font color field.

The background color field works the same way as the font color field. If you want to revert the default background color, you can copy and paste #19181A into the font color field.
